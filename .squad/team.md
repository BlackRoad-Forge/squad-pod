## Project Context

- **Project:** squad-pod
- **User:** Brian Swiger
- **Created:** 2026-03-05
- **Universe:** The Simpsons
- **Stack:** VS Code Extension (TypeScript, esbuild), React 19 webview (Vite, Canvas 2D pixel art), Squad integration (.squad/ state reading)
- **Inspiration:** pablodelucca/pixel-agents (animated pixel art office for AI agents), adapted for bradygaster/squad
- **Description:** A VS Code extension that renders an animated pixel art office scene where Squad AI agents appear as characters. Reads .squad/ directory state to show agent activity in real-time — who's working, what decisions are being made, and the overall health of the team.
